## About The Project

Phát triển hệ thống AI Agent đa kênh tích hợp kỹ thuật RAG ứng dụng trong quy trình chăm sóc khách hàng tự động

Kết quả của dự án sẽ được chúng tôi thể hiện trên nền tảng website Streamlit - với khả năng cho người dùng tương tác dữ liệu để mô phỏng quá trình thực hiện trò chuyện với nhân viên chăm sóc khách hàng trên hệ thống.
